# TowingNo.1 - Professional Towing Website

A premium, high-performance Next.js 16 website for TowingNo.1, a 24/7 emergency towing and roadside assistance company serving British Columbia and the Lower Mainland.

- **Premium Navy & Amber Design**: Professional, high-contrast aesthetic using a deep navy and warm gold palette.
- **Vector Geometric Logo**: Modern CSS/SVG based logo for crystal clear rendering at any size.
- **Glassmorphism UI**: Modern frosted-glass effects, ambient light orbs, and subtle depth.
- **Fully Responsive**: Mobile-first design that transitions seamlessly between mobile, tablet, and desktop.
- **SEO Optimized**: Complete metadata, Open Graph, sitemap, robots.txt, and localized schema data.
- **Fast Performance**: Next.js App Router with optimized image delivery and minimal JS overhead.
- **Framer Motion Animations**: Smooth staggered reveals, floating elements, and interactive transitions.
